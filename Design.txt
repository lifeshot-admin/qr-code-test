[서비스 명: 치이즈(CHIIZ) - 포토그래퍼 전용 웹앱]

1. 서비스의 목적
- 투어 현장에서 포토그래퍼가 고객의 예약을 확인하고, 인증사진을 찍어 버블 DB에 실시간으로 업로드하는 관리 도구.

2. 주요 화면 및 사용자 흐름 (UI Flow)
- 스캔 화면: 카메라를 통해 고객의 QR 코드를 인식하거나, 수동으로 예약 화면을 촬영함.
- 확인 화면: 인식된 예약 ID가 버블 DB(pose_reservation)에 존재하는지 확인하고 정보를 표시함.
- 인증샷 화면: 고객의 얼굴이 포함된 인증 사진을 촬영함.
- 완료 화면: 촬영된 인증샷을 버블 DB(auth_photo)에 업로드하고, 해당 고객에게 배정된 'Reserved_pose' 가이드 목록을 불러와서 보여줌.

3. 기술적 특징
- 백엔드: Bubble.io의 Data API를 사용하여 모든 데이터를 실시간 CRUD 처리함.
- 프론트엔드: Next.js와 TypeScript를 사용하여 모바일 브라우저 환경에서 고성능 카메라 제어를 구현함.
- 통신 규격: 버블에서 제공하는 Swagger API 명세서를 준수함.
